# 火源灵感火花 · 小程序使用说明书

> 本文档依据当前小程序功能整理，产品定位为 **AI 驱动的智能创作平台**（全局标题亦为「火源文案」）。底部三个主 Tab：**发现**、**IP 操作台**、**我的**。

---

## 一、快速上手

1. 打开微信内本小程序，底部切换三个主栏目即可使用核心功能。
2. **部分能力需登录**（手机号快捷登录）：如完整 IP 创建流程、我的灵感等；未登录时部分页面会提示前往登录，也可选择「先去逛逛」浏览首页。
3. **会员与算力**：非普通会员可查看「我的算力」并充值；会员权益以「开通会员」页及后台配置为准。

## 三、IP 操作台

**路径**：底部 Tab「IP 操作台」

IP 操作台围绕「**项目**」组织：每个项目有独立 **IP 人设** 与内容风格，用于操盘与创作。

### 3.1 无项目时

- 进入 **空状态**，可按引导 **创建项目**，进入 **IP 信息定位调研**（多步表单）。

### 3.2 已有项目但未选中当前项目时

- 展示 **项目列表**，请选择要操盘的项目；也可通过顶部等区域 **切换项目**（抽屉列表）。

### 3.3 已选中当前项目时（操作台主页）

主界面包含：

| 区域 | 说明 |
|------|------|
| 顶部栏 | 显示当前项目名称、用户信息与积分等；可 **切换项目**。 |
| 当前人设卡片 | 展示当前项目人设；点击可进入 **微调人设**（与创建时相同的分步调研页，编辑模式保存后更新项目）。 |
| **今天拍点啥** | 分类网格，数据来自后台「快捷入口」。点击行为由配置决定：常见为跳转 **AI 文案生成**（带智能体、指令说明）或小程序内其他页面（如热点页）；部分类型若未开放会提示「功能开发中」。 |
| **快捷指令库** | 快捷指令卡片列表，通常进入 **AI 文案生成** 页，并带上对应智能体与预设指令上下文。 |
| **历史对话** | 展示与当前业务相关的会话列表；点击某条会进入 **AI 文案生成** 并带上会话 / 智能体信息以便继续沟通。 |
| **下拉刷新** | 刷新历史对话与快捷入口数据。 |
| 右下角 **悬浮按钮** | 打开 **灵感捕捉**：输入文字、可选标签，保存到「我的灵感」（可关联当前项目）；语音输入若提示即将上线，以实际版本为准。 |

### 3.4 创建项目 / IP 调研流程

**创建新项目**时，进入多步引导（步骤可能包含）：

1. **设定身份** — 行业、角色等基础定位  
2. **注入灵魂** — 价值观、故事感等人格侧信息  
3. **定义风格** — 语气、表达习惯等  
4. **激活大脑** — 汇总与确认  

填写过程中支持 **自动缓存**（意外退出可再次进入时选择继续或清空）。**新建**完成最后一步会 **生成 IP 定位报告**，随后在报告页可 **保存为项目**；**编辑 / 微调人设**则在最后一步 **保存人设更新** 并返回。

> 未登录时进入创建页会提示登录，不阻断进入，但完整保存与云端能力需登录。

### 3.5 相关子页面（从操作台或配置进入）

- **IP 定位报告**：查看生成结果与保存项目。  
- **AI 文案生成**：与智能体对话生成文案，可继承分类 / 快捷指令 / 历史会话上下文。  
- **AI 对话**：发现页「全能对话」等入口使用的独立对话能力（与文案页分工以实际产品为准）。  
- **历史对话**（列表页）：集中查看会话列表（与操作台内嵌列表互补，以入口为准）。  
- **抖音热点榜单**：查看热点列表，可对条目使用「蹭热点」等操作（与创作流程联动）。

## 四、我的

**路径**：底部 Tab「我的」

### 4.1 账号与资料

- **登录**：未登录显示登录按钮，进入 **手机号快捷登录**；需勾选同意《用户协议》《隐私政策》。  
- **头像**：在微信小程序内可通过 **微信头像** 能力选择并上传更新。  
- **手机号**：登录后展示脱敏手机号。  
- **会员标识**：若为非普通用户，展示等级与到期时间等（以实际字段为准）。

### 4.2 会员与算力

- **开通会员**：普通用户可见升级卡片，进入 **开通会员** 了解权益与购买。  
- **我的算力**（非普通用户展示）：显示算力点余额，可进入 **算力明细**、**充值算力**。

### 4.3 功能菜单

| 入口 | 说明 |
|------|------|
| **我的灵感** | 需登录。查看、搜索、筛选已保存的灵感；支持归档、置顶等操作；可从灵感发起生成或对话（以页面按钮为准）。 |
| **我要推荐** | 邀请好友等活动说明与参与入口。 |
| **联系客服** | 咨询会员、使用问题等。 |

**下拉刷新「我的」页**可刷新用户信息（算力、会员状态等）。

## 六、常见问题与提示

1. **提示「功能开发中，请耐心等待」**  
   部分菜单或配置指向尚未发布的页面，或该快捷类型（如部分 skill / prompt）仍在建设中。

2. **发现页与操作台的「对话」差异**  
   发现页快捷入口多跳转 **AI 对话**；操作台内分类、快捷指令、历史对话多跳转 **AI 文案生成**，并会带上项目 / 指令 / 会话上下文，更适合围绕当前 IP 创作。

3. **网络与加载失败**  
   操作台若加载失败，页面提供 **重试**；仍失败请检查网络或稍后重试。

4. **算力消耗**  
   各 AI 功能是否扣减算力、扣减规则以服务端与会员策略为准，请在「我的算力」「充值」页查看最新说明。
